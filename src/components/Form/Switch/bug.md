# Switch 开关组件 Bug 记录与修改情况

| Bug 描述                                        | 根本原因分析                                                                                    | 修复方案与改动细节                                                                                                                                                   | 状态       |
| :---------------------------------------------- | :---------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------- |
| 表单 reset 时 Switch 没有自动更新状态并清空校验 | Switch 原先绑定的触发器设置不当，在重置或者表单级校验触发时，表单收集不到正确的 `change` 通知。 | 在使用 `useFormControl` 时，明确设置 triggers 的具体参数 `{ input: false, blur: false, change: "change" }`，让状态通知在 `change` 事件中统一流向 FormItem 校验管道。 | **已修复** |
| 键盘触发切换时会导致浏览器默认的页面滚动        | 在监听 keydown 的 `Space` 时，如果不阻止事件默认行为，浏览器会默认进行整页下滚。                | 在 `onKeyDown` 键盘监听逻辑中，增加 `e.preventDefault()` 阻止空格引起的默认页面滚动，只响应状态切换。                                                                | **已修复** |
